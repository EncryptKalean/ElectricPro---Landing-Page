# ⚡ ElectricPro - Landing Page para Serviços Elétricos

## Sobre o Projeto

**ElectricPro** é uma landing page moderna e responsiva desenvolvida com **HTML, CSS e JavaScript puro**.
O projeto simula um site profissional para prestadores de serviços elétricos, focado em captação rápida de clientes através de um formulário integrado ao WhatsApp.
O objetivo é demonstrar habilidades de desenvolvimento front-end criando uma página simples, funcional e orientada à conversão, ideal para negócios locais.

O projeto destaca-se por:

- Layout moderno e profissional.
- Estrutura leve e otimizada, sem uso de frameworks.
- Design totalmente responsivo.
- Integração direta com WhatsApp para contato rápido.
- Boas práticas de acessibilidade e semântica HTML.

## 💬 Formulário Inteligente

Um dos focos principais da landing page é o formulário de contato otimizado para conversão.
Ele coleta informações essenciais do cliente:

- Nome
- Telefone
- Descrição do problema

Ao enviar, o sistema gera automaticamente uma mensagem e abre o WhatsApp do prestador de serviço com os dados preenchidos.

Isso permite:

- Contato imediato com o cliente
- Redução de fricção no atendimento
- Maior taxa de conversão

---

## 🎨 Interface

O design foi pensado para transmitir profissionalismo e confiança, elementos essenciais para serviços técnicos.

Principais características visuais:

- Card com efeito glassmorphism
- Botões com destaque visual
- Layout centralizado para foco no formulário
- Background fixo com efeito parallax leve, criando sensação de profundidade durante o scroll
- Background com textura sutil para dar profundidade

---

## 📱 Responsividade

A landing page adapta-se automaticamente para diferentes dispositivos:

- Desktop
- Tablet
- Smartphone

Todos os elementos foram projetados para manter boa usabilidade em telas menores, garantindo uma experiência consistente.

## ♿ Acessibilidade

O projeto também considera práticas básicas de acessibilidade:

- Uso de HTML semântico
- Labels associadas corretamente aos inputs
- Uso adequado de ARIA em elementos SVG
- Navegação clara e estrutura organizada

---

## 🛠 Tecnologias Utilizadas

- HTML5 – Estrutura semântica.
- CSS3 – Layout moderno, responsivo e efeitos visuais.
- JavaScript (Vanilla) – Lógica do formulário e integração com WhatsApp.

Nenhuma biblioteca externa foi utilizada, mantendo o projeto leve e didático.


## 🎯 Objetivo do Projeto

Este projeto foi criado para:
- Praticar desenvolvimento front-end moderno
- Demonstrar habilidades em HTML, CSS e JavaScript
- Servir como exemplo de landing page para negócios locais
- Compor portfólio profissional
